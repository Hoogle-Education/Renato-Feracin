## Estrutura de uma classe 

```
 1. atributos
  1.1 estaticos/finais
  1.2 atributos próprios
 2. construtores
  2.1 construtor padrão (se necessário)
  2.2 construtores personalizados (se necessário)
 3. getters and setters
 4. métodos
 5. toString
 6. equals and hashcode
```
